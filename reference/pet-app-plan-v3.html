PET 词汇特训 · MVP V3

从第一性原理出发，砍掉一切非必要，只做一件事：让孩子真正记住单词

第一性原理推导

公理 1：PET 听力、阅读、写作、口语的本质瓶颈是同一个——词汇量不够且不牢固。认识 500 个词但每个都要想 3 秒的孩子，阅读和听力一定差。认识 200 个词但每个都秒懂的孩子，阅读和听力至少及格。

公理 2：一个词被"真正掌握"需要三道神经回路全部打通——眼睛看到立刻知道意思、耳朵听到立刻知道意思、想到意思能写出单词。只打通一条（比如看到认识）是不够的。

公理 3：孩子每天愿意花在课外学习上的注意力是有限的。10-15 分钟是可持续的，超过这个时间会产生抵触，反而损害长期坚持。

→ 因此 MVP 只做一件事：每天 5 个词，三道神经回路同时训练，用间隔重复对抗遗忘。不做阅读题、不做听力题、不做错题本——这些是 V1.1 的事。在词汇没扎实之前，做任何阅读理解都是在练习"猜"，而不是练习"读"。

V2 → V3：砍掉了什么，为什么

V2 MVP 功能V3 决策第一性原理理由
词汇阅读（10 篇短文 + 理解题） 砍掉 每个单词卡片自带一个例句，例句本身就是"微型阅读"。20 秒读完一句比 5 分钟读一篇短文更高效。短文阅读理解放在 V1.1，等词库积累到 200+ 再上。
词汇听力（10 段对话 + 理解题） 砍掉 "听音选词"测验本身就是听力训练——浏览器朗读一个单词，孩子从 4 个选项中选。这已经训练了音→义的神经回路。对话听力同样需要词库积累，放在 V1.1。
单独的错题本模块 砍掉 间隔复习队列天然就是错题本——答错的词自动高频复习，答对的词自动降低频率。不需要一个单独的"浏览错题"界面。
复杂的进度图表 砍掉 MVP 只保留三个数字：累计词汇量、连续打卡天数、今日完成状态。V1.1 再加正确率曲线。
成就徽章系统 保留但精简 只保留 50/100/200 词里程碑和 7 天连续打卡两个最简单的成就。不搞复杂的积分等级。
闪卡学习 + 三通路测验 保留 + 强化 这是核心引擎，不能砍。而且要加强——闪卡融入发音、例句朗读，测验加入响应时间追踪（卡壳超过 3 秒也算需要复习）。
智能复习队列 保留 + 简化 从 1/2/4/7/15 天间隔简化为 4 个等级：当天→明天→3天后→7天后。对孩子来说足够，对家长来说可理解。
— 新增 例句朗读 + 跟读提示。每个单词的例句都可以点击播放朗读，并提示孩子跟读。在 MVP 中就把"口语发音"的种子埋下，但不做录音评分。

5 个阶段，总计约 12 分钟。孩子打开应用 → 自动进入流程 → 做完关闭。

阶段一：学新词（约 3 分钟）

  • 每天推送 5 个新词，属于同一个话题（如"食物"或"旅行"）
  • 闪卡模式：正面显示英文 + 自动播放发音，点按翻转显示中文释义 + 例句
  • 例句也会自动朗读一遍，孩子可以看到这个词在句子中怎么用
  • 底部有"下一个"按钮和一个"跟读"按钮（点击再读一次例句）
  • 5 个词过一遍约 3 分钟，每个词约 35 秒

阶段二-四：三通路测验（约 6 分钟）

👀 看义选词（2 min）

显示一个英文单词，从 4 个中文选项中选出正确释义。训练"形→义"通路。这是最轻松的，放在第一轮帮孩子建立信心。5 题 × 约 20 秒/题。

👂 听音选词（2 min）

浏览器朗读一个单词（不显示文字），从 4 个英文选项中选出听到的词。训练"音→形"通路。这是孩子最弱的一环，放在第二轮。5 题 × 约 20 秒/题。

✍️ 拼写填空（2 min）

显示中文释义 + 例句（缺目标单词），听发音后输入拼写。训练"义→拼写"通路。这是最难的，放在最后。5 题 × 约 25 秒/题。

测验覆盖今天 5 个新词。任何一轮答错的词，自动进入"待复习队列"。响应超过 3 秒的题也算"需要加强"。

阶段五：复习旧词（约 3 分钟）

  • 从复习队列中取出今天该复习的词（通常 5-8 个）
  • 混合题型快速过一遍——每个词随机出一种题型（看义/听音/拼写）
  • 答对 → 降低复习频率（晋级到下一级间隔），答错 → 重置到今天
  • 复习完显示"今日学习完成！"和三个关键数字

四级间隔复习算法

等级含义下次复习时间如何晋级
L0 · 未掌握 今天刚学，或之前答错过 当天（立即重新加入今日复习） 答对一次 → L1
L1 · 短时记忆 刚记住，还不稳定 明天 答对 → L2
L2 · 巩固中 隔天还能记住 3 天后 答对 → L3
L3 · 已掌握 过几天还能记住 7 天后 答对 → 保持 L3（7天后）

任何等级答错 → 直接掉回 L0，当天重新复习。这个机制确保真正薄弱词不会被遗漏。

MVP V3 · 交付清单

#功能描述
1首页 Dashboard三个数字：累计词汇量、连续打卡天数、今日状态（未开始/进行中/已完成）。一个按钮：开始学习。
2闪卡学习5 张卡片，自动翻页。正面英文 + 自动发音，背面中文 + 例句 + 例句朗读。每张卡片停留由孩子自己控制（点"下一个"）。
3看义选词测验5 题。显示一个英文词 + 4 个中文选项。选对显示绿色勾 + 例句朗读；选错显示红色叉 + 正确答案。
4听音选词测验5 题。浏览器朗读一个词（可重播），4 个英文选项。选对选错同上。每道题自动记录响应时间。
5拼写填空测验5 题。显示中文 + 例句（单词挖空），听发音，输入拼写。不区分大小写。允许一次错误后显示首字母提示。
6智能复习新词测验结束后自动进入复习。从队列中取出今天该复习的词，随机出题。答对晋级，答错重置。
7完成页今日收获总结：学会了 N 个词、复习了 M 个词、正确率。一句鼓励的话。连续打卡天数更新。
8词库内置 200 个 PET 核心词，分 10 个话题（食物、家庭、学校、运动、旅行、天气、健康、购物、动物、日常）。按话题顺序推送。
9localStorage所有数据本地持久化。刷新/关闭不丢失。

UI 设计原则（针对 10-15 分钟场景）

  • 一次只做一件事。屏幕上永远只有一个核心交互元素，不出现多个按钮争夺注意力。
  • 全屏沉浸。闪卡和测验都是全屏显示，没有顶部导航栏、底部 Tab 栏来分散注意力。进入学习模式后，只有"返回"和"下一个"两个出口。
  • 即时反馈 + 不打断。选对选错后立即显示结果（0.5 秒动画），然后自动进入下一题，不需要点"确认"。
  • 手指友好。所有可点击区域至少 44×44px，按钮间距足够大。适合孩子可能在平板上使用。
  • 暖色调 + 圆角 + 大字体。避免冷冰冰的工具感，让应用看起来像一本"会说话的单词书"。
  • 无惩罚、无倒计时。测验不计时（后台默默记录响应时间但不显示），答错不扣分，鼓励尝试。
